# Notice

Crayon Crew Illustrations is an agent skill inspired by two sources. Please keep this attribution when redistributing or adapting this repository.

## Workflow inspiration

[ian-xiaohei-illustrations](https://github.com/helloianneo/ian-xiaohei-illustrations) by Ian ([helloianneo](https://github.com/helloianneo)).

That project pioneered the Codex skill pattern for article illustrations: digest the text, produce a shot list, invent one metaphor per image, require a recurring character to perform the core action, then QA and deliver. This skill reuses that workflow structure in English and makes it agent-agnostic.

Ian Xiaohei Illustrations is MIT-licensed. The "小黑" character and its Chinese visual language belong to Ian's work and are not reused here.

## Visual style inspiration

[tubik.arts](https://dribbble.com/tubik_arts) on Dribbble.

The cream / charcoal / marigold palette, grainy crayon texture, bold black-and-white patterns, oversized props, and playful character energy that define this skill's look are inspired by Tubik's illustration work. The Crayon Crew characters and all style-reference images bundled in this repository are original designs generated for this skill — no Tubik artwork is redistributed here.
